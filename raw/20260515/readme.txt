R.E.P.O.'s untranslated localization strings are located in these Tab-Separated Value (TSV) files (HUD.tsv, Menu.tsv and Game.tsv).

If you would like to manually translate the game into another language, please copy these files into the parent "StreamingAssets/Localizations/" folder.

The game will automatically load strings from the TSV files in the "StreamingAssets/Localizations/" folder and update the localization tables at runtime.
